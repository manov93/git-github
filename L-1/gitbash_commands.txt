mkdir -> to make directory/folder
cd -> to change directory
ls -> to list files/folder within the directory
touch -> to create files
git status -> to cheek's git tracking
git init -> to start observing
git add -> to add (./-a)
git commit ->to commit on new files
git commit -m "meaningful msg" -> with -m"___" we can add msg with commit
git log -> to check log of the files
git checkout ID -> to review the previous commit and momdification on the files
git checkout main -> switch to current version
git branch -> to get list of sub branch or any features files.
git branch [NAME] -> to create a new branch
git branch -d -> to delete the branch (from main head)
git checkout -b [File NAME] -> this command will create new branch and switch to it.
